# -Ganesha-s-Pattern

Take as input N, an odd number (>=5) . Print the following pattern as given below for N = 7.

*  ****
*  *
*  *
*******
   *  *
   *  *
****  *
Input Format:
Enter value of N ( >=5 )

Constraints:
N >= 5

Output Format

Sample Input
7

Sample Output
*  ****
*  *
*  *
*******
   *  *
   *  *
****  *
